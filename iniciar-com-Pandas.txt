# Como você pode começar a usar o Pandas? 
## Noções básicas essenciais explicadas!

Pandas é uma biblioteca Python usada para manipulação e análise de dados.

Ele oferece muitos recursos, incluindo:

I. Estruturas de dados
Series (matriz rotulada 1D) e DataFrame (tabela rotulada 2D).

II. Manipulação e análise de dados
Suporta operações como mesclagem, junção, remodelação,
agregação e filtragem.

III. Tratamento de dados ausentes
Oferece ferramentas robustas para detectar, encher e 
removendo dados ausentes.

IV. Entrada/Saída e Integração
 Facilita a leitura e gravação de dados de vários
formatos de arquivo (CSV, Excel, SQL).

## As operações básicas em Pandas incluem:

✓ Criando uma série
↳ 𝗽𝗱.𝗦𝗲𝗿𝗶𝗲𝘀()

✓ Criando um DataFrame
↳ 𝗽𝗱.𝗗𝗮𝘁𝗮𝗙𝗿𝗮𝗺𝗲()

✓ Leitura de dados de um arquivo
↳ 𝗽𝗱.𝗿𝗲𝗮𝗱_𝗰𝘀𝘃()
↳ 𝗽𝗱.𝗿𝗲𝗮𝗱_𝗲𝘅𝗰𝗲𝗹()
↳ 𝗽𝗱.𝗿𝗲𝗮𝗱_𝘀𝗾𝗹()

✓ Visualizando dados
↳ 𝗵𝗲𝗮𝗱()
↳ 𝘁𝗮𝗶𝗹()

✓ Obtendo informações sobre o DataFrame
↳ 𝗶𝗻𝗳𝗼()
↳ 𝗱𝗲𝘀𝗰𝗿𝗶𝗯𝗲()

✓ Seleção de dados
↳ 𝗱𝗳['𝗡𝗮𝗺𝗲']
↳ 𝗱𝗳[['𝗖𝗶𝘁𝘆', '𝗦𝗮𝗹𝗮𝗿𝘆']]
↳ 𝗱𝗳.𝗶𝗹𝗼𝗰[𝟬]
↳ 𝗱𝗳[𝗱𝗳['𝗦𝗮𝗹𝗮𝗿𝘆'] > 𝟱𝟬𝟬𝟬𝟬]

✓ Adicionando novas colunas
↳ 𝗱𝗳['𝗘𝘅𝗽𝗲𝗿𝗶𝗲𝗻𝗰𝗲'] = [𝟭, 𝟯, 𝟰, 𝟱]

✓ Apagando de colunas
↳ 𝗱𝗳 = 𝗱𝗳.𝗱𝗿𝗼𝗽('𝗦𝗮𝗹𝗮𝗿𝘆', 𝗮𝘅𝗶𝘀=𝟭) 

✓ Apagando linhas
↳ 𝗱𝗳 = 𝗱𝗳.𝗱𝗿𝗼𝗽(𝟭, 𝗮𝘅𝗶𝘀=𝟬) 

✓ Classificação de dados
↳ 𝗱𝗳 = 𝗱𝗳.𝘀𝗼𝗿𝘁_𝘃𝗮𝗹𝘂𝗲𝘀('𝗔𝗴𝗲')
↳ 𝗱𝗳 = 𝗱𝗳.𝘀𝗼𝗿𝘁_𝘃𝗮𝗹𝘂𝗲𝘀(['𝗖𝗶𝘁𝘆', '𝗔𝗴𝗲'], 𝗮𝘀𝗰𝗲𝗻𝗱𝗶𝗻𝗴=[𝗧𝗿𝘂𝗲, 𝗙𝗮𝗹𝘀𝗲])

✓ Lidando com dados ausentes
↳ 𝗱𝗳['𝗦𝗮𝗹𝗮𝗿𝘆'] = 𝗱𝗳['𝗦𝗮𝗹𝗮𝗿𝘆'].𝗳𝗶𝗹𝗹𝗻𝗮(𝘃𝗮𝗹𝘂𝗲=𝗻𝗽.𝗺𝗲𝗮𝗻(𝗱𝗳['𝗦𝗮𝗹𝗮𝗿𝘆']))
↳ 𝗱𝗳 = 𝗱𝗳.𝗱𝗿𝗼𝗽𝗻𝗮()

✓ Renomeando colunas
↳ 𝗱𝗳.𝗿𝗲𝗻𝗮𝗺𝗲(𝗰𝗼𝗹𝘂𝗺𝗻𝘀={'𝗡𝗮𝗺𝗲': '𝗙𝘂𝗹𝗹 𝗡𝗮𝗺𝗲'}, 𝗶𝗻𝗽𝗹𝗮𝗰𝗲=𝗧𝗿𝘂𝗲)

✓ Manipulação de duplicatas
↳ 𝗱𝗳 = 𝗱𝗳.𝗱𝗿𝗼𝗽_𝗱𝘂𝗽𝗹𝗶𝗰𝗮𝘁𝗲𝘀(𝘀𝘂𝗯𝘀𝗲𝘁=['𝗡𝗮𝗺𝗲', '𝗖𝗶𝘁𝘆'], 𝗸𝗲𝗲𝗽='𝗹𝗮𝘀𝘁')

✓ Plotagem básica com Matplotlib

Aprender esses princípios básicos do Pandas ajuda você a manipular
e analisar melhor os dados em Python.
